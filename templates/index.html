CASPER соберёт для вас препринты научных статей по их названиям


Чтобы скачать нужные вам статьи нужно выполнить несколько простых шагов.

Входные данные


В это окошко вы можете вставить список публикаций, которые вы хотите скачать. Одна строчка - одна публикация. Можете указывать только название статьи, или (чтобы избежать ложных срабатываний) добавить через знак табуляции коллектив авторов. Для этого достаточно скопировать и вставить сюда один (или два) столбика из Excel, Numbers или OpenOffice.

Получение данных о статьях

Нажав на кнопку ниже вы запустите сбор данных о ваших статьях с сайта arxiv.org. Из-за особенностей пользовательского соглашения, каждая на каждый заголовок из вашего списка будет уходить 3-5 секунд. Приготовьтесь немного подождать. После окончания перед вами появится таблица со статьями, а также скрипт для самостоятельного скачивания.


Скрипт для самостоятельного скачивания статей в PDF. Если вам нужны оригинальные PDF статей, тогда вставьте этот текст в консоль MacOS или Linux. Он скачает вас все препринты в PDF и разместит их папки по годам.
